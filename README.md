# Log-Tool

## About
This tool is used for converting and combining Hoverfly drone data logs to easier veiw in Mission Planner

##  Requirements (Source)
* Python 3.5+
* JDK 15.0.1+
* PySimpleGui 4.32.1+
* Mechanical Soup 1.0+
* Winping 0.2.4+

## Requirements (Executable)
* JDK 15.0.1+ (automatically installed)
* Windows 10
